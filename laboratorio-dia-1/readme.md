# Calculadora en JavaScript

Este es un proyecto de una calculadora simple hecha con **HTML, CSS y JavaScript**. Permite realizar operaciones básicas como suma, resta, multiplicación y división de dos números ingresados por el usuario.

## 📌 Características

- Interfaz sencilla y fácil de usar.
- Validación de entrada para evitar errores.
- Deshabilita la división cuando el segundo número es 0.
- Botón para limpiar los campos y el resultado.

## 🚀 Tecnologías utilizadas

- **HTML**: Para la estructura del proyecto.
- **CSS**: Para el diseño y estilos.
- **JavaScript**: Para la lógica de las operaciones matemáticas.

## 📋 Instrucciones de uso

1. Ingresa dos números en los campos de entrada.
2. Presiona el botón correspondiente a la operación que deseas realizar.
3. El resultado aparecerá en la pantalla.
4. Si deseas borrar los valores, presiona el botón **"Limpiar"**.
